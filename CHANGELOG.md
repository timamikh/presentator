# Changelog

Формат: изменения по версиям в стиле Keep a Changelog (упрощённо).

## [Unreleased]

### Added

- Расширенные настройки дизайна в UI:
  - «Дизайн презентации»: палитра 2–5 цветов с ролями, style tags, кастомный дизайн‑промпт
  - «Дизайн текстов»: 2–5 текстовых стилей (цвет/шрифт/начертание/кегль + “где используется”)
- Генерация блока `DESIGN_BRIEF` в n8n (node `Build LLM Prompt`) и включение его в `userMessage` (включая refinement).
- Усиленные “anti-minimalism” правила в `DESIGN_BRIEF` (квоты использования палитры, обязательные дизайн‑компоненты).

### Changed

- `Build LLM Prompt`: вместо «сырых» legacy настроек в промпт добавляется нормализованный `DesignDataJSON` (меньше шумных полей → лучше следование дизайну).
