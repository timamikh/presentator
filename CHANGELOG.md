## [0.2.0] — 2026-03-17

### Added

- Пользовательское хранилище вложений (страница `Хранилище`), загрузка/поиск/фильтрация, редактирование промптов, удаление.
- Добавление вложений из хранилища при создании презентации + отдельные промпты на каждое вложение (и для новых загрузок, и для выбранных из хранилища).
- Поддержка изображений в слайдах: LLM может размещать изображения в HTML (через `src="attachment:<id>"`), далее подстановка в `data:` URL.
- «Уточнения» (refinement) после генерации: чат-запрос к LLM с полным контекстом + история версий + восстановление версии.

### Changed

- n8n workflow: мультимодальный `userContent` (текст + изображения), подстановка `attachment:<id>` в HTML после ответа LLM.
- API: увеличены лимиты JSON body (для payload с base64).
- Улучшена устойчивость фронтенда: безопасное чтение токена из `localStorage` для `<img>` превью, восстановление polling после уточнений.

### Database

- Новые таблицы: `presentator.attachments`, `presentator.job_attachments`, `presentator.job_revisions`.
